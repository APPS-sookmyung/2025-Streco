## 📖 프로젝트 소개

### 1️⃣ 서비스 소개

방송 속 소중한 순간을 또 찾아보고 싶은데, 놓친 적 있으신가요? Streco가 그 고민을 해결해 드리겠습니다. <br/>
지나간 장면은 돌아오지 않아요! 스트리밍 속 중요한 순간을 실시간으로 기록하고 언제든지 다시 확인해보세요. 이미 방송이 시작된 것 같은데요? **Streco**를 사용할 때가 왔습니다!

<br/>

 🚀 배포 링크: https://2025-streco.vercel.app/
 
---

### 2️⃣ 주요 기능 (Key Features)

🔴 **스트리머 관리하기** <br/>
응원하는 프로 게이머와 방송인을 빠르게 등록하고 삭제하며 효율적으로 관리할 수 있습니다.

🔴 **방송 일정 등록하기** <br/>
등록한 방송은 캘린더에 하이라이트로 표시되어, 스케줄을 한눈에 확인할 수 있습니다.

🔴 **하이라이트 기록하기** <br/>
중요한 순간을 실시간으로 기록하세요. Streco가 초 단위로 저장해, 빠른 다시보기를 지원합니다.
